** Función: calcular tasas de inflación

   #+NAME: fn_inflacion
   #+BEGIN_SRC R
   tasa_inflacion <- function(serie) {
     n <- length(serie)
     c(NA, (serie[2:n] - serie[1:(n-1)]) / serie[1:(n-1)] * 100)
   }

   # Aplicar a todas las columnas de países
   paises <- c("canada","francia","alemania","italia","japon","ru","eeuu")
   inflacion <- ipc
   for (p in paises) {
     inflacion[[p]] <- tasa_inflacion(ipc[[p]])
   }
   inflacion <- inflacion[!is.na(inflacion$canada), ]  # quitar 1973
   print(round(inflacion, 2))
   #+END_SRC

** Estadísticos de variabilidad

   #+BEGIN_SRC R
   variabilidad <- data.frame(
     pais   = paises,
     media  = sapply(paises, \(p) round(mean(inflacion[[p]], na.rm=TRUE), 2)),
     desv   = sapply(paises, \(p) round(sd(inflacion[[p]],   na.rm=TRUE), 2)),
     maximo = sapply(paises, \(p) round(max(inflacion[[p]],  na.rm=TRUE), 2))
   )
   variabilidad$cv <- round(variabilidad$desv / variabilidad$media * 100, 1)

** Conclusiones 1.2

   1. *Co-movimiento general*: todas las series descienden en paralelo
      desde inicios de los 80 → factor externo común (OPEP, política
      monetaria restrictiva).
   2. *Diferencial de nivel*: Italia y G.B. sobre EE.UU.; Alemania y
      Japón por debajo en casi todo el período.
   3. *La anomalía japonesa*: deflación a partir de 1999-2002 →
      trampa de liquidez tras el colapso de la burbuja de activos.
   4. *Causalidad*: el co-movimiento NO implica que EE.UU. cause la
      inflación del resto. Causas alternativas:
      - Choques de oferta comunes (petróleo)
      - Interdependencia financiera (tasas de interés)
      - Comercio internacional de bienes transables
      El test formal es el de causalidad de Granger (cap. 21).

* Notas técnicas

** Fórmula LaTeX de referencia

   La tasa de inflación entre $t-1$ y $t$:
   $$\pi_t = \frac{\text{IPC}_t - \text{IPC}_{t-1}}{\text{IPC}_{t-1}} \times 100$$

   Aproximación logarítmica (válida para $\pi_t$ pequeño):
   $$\pi_t \approx \left[\ln(\text{IPC}_t) - \ln(\text{IPC}_{t-1})\right] \times 100$$
