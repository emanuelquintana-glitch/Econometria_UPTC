** Función reutilizable: tasa de inflación

   #+NAME: fn_inflacion
   #+BEGIN_SRC R
   tasa_inflacion <- function(serie) {
     n <- length(serie)
     c(NA, (serie[2:n] - serie[1:(n-1)]) / serie[1:(n-1)] * 100)
   }

   paises <- c("canada","francia","alemania","italia","japon","ru","eeuu")
   inflacion <- ipc
   for (p in paises) inflacion[[p]] <- tasa_inflacion(ipc[[p]])
   inflacion <- inflacion[!is.na(inflacion$canada), ]   # desde 1974
   print(round(inflacion, 2))
   #+END_SRC

** Estadísticos de variabilidad

   variabilidad <- data.frame(

** Conclusiones

   1. Co-movimiento general: descenso paralelo desde los 80 → factor externo
      común (OPEP, política monetaria restrictiva de la Fed/Bundesbank).
   2. Italia y G.B. sistemáticamente sobre EE.UU.; Alemania y Japón por debajo.
   3. Anomalía japonesa: deflación desde 1999 → trampa de liquidez post-burbuja.
   4. *Causalidad*: el co-movimiento NO implica que EE.UU. cause la inflación
      del resto. Causas alternativas: choques de oferta comunes, integración
      financiera, comercio de bienes transables.
      Test formal → causalidad de Granger (cap. 21).

* Notas técnicas

** Fórmulas clave

   Tasa de inflación exacta:
   $$\pi_t = \frac{\text{IPC}_t - \text{IPC}_{t-1}}{\text{IPC}_{t-1}} \times 100$$

   Aproximación logarítmica (válida para $\pi_t$ pequeño):
   $$\pi_t \approx \left[\ln(\text{IPC}_t) - \ln(\text{IPC}_{t-1})\right] \times 100$$

   Test de causalidad de Granger (cap. 21):
   $$\pi_t^j = \sum_{k=1}^p \alpha_k \pi_{t-k}^j
             + \sum_{k=1}^p \beta_k \pi_{t-k}^{\text{US}}
             + \varepsilon_t, \qquad H_0: \beta_1 = \cdots = \beta_p = 0$$
