# Manual Técnico de Econometría: Multicolinealidad y Modelos de Producción

Este documento detalla los métodos de detección, corrección y la aplicación práctica de restricciones en modelos de regresión múltiple, basado en el análisis de la función de producción de Dougherty (1899-1922).

---

## 1. Detección de Multicolinealidad

La multicolinealidad es la existencia de una relación lineal perfecta o exacta entre las variables explicativas.

### 1.1. Técnica de Thails (Coeficiente $m$)

Se calcula evaluando el porcentaje de contribución de las variables explicativas respecto a la dependiente, contrastando el $R^2$ global frente a las contribuciones marginales.

**Fórmula:**
$$m = R^2 - \sum_{i=1}^{k} (R^2 - R^2_{h \neq i})$$

* **Lógica:** Si al sacar una variable el $R^2$ no cae significativamente ($R^2 - R^2_h \approx 0$), la variable es redundante.
* **Análisis de dependencia:**
  * **Se elimina la dependiente:** Se busca colinealidad solo entre las explicativas ($X_{2t}, X_{3t}, X_{4t}$).
  * **Se mantiene la dependiente:** Se analiza cuánto explica cada variable respecto a $Y$.

### 1.2. Síntomas Clásicos en la Práctica

1. **$R^2$ muy elevado (ej. 0.97)** con **insignificancia individual** de los parámetros (Estadísticos $t$ bajos o con signos erróneos).
2. **Signos de coeficientes contrarios a la teoría económica:** En una Cobb-Douglas, se espera que el Capital ($K$) tenga signo positivo (elasticidad positiva). Un signo negativo es evidencia fuerte de colinealidad.

---

## 2. Métodos para Disminuir y Corregir el Grado de Colinealidad

### 2.1. Acciones Inmediatas

1. **Aumentar el tamaño de la muestra ($n$):** Reduce la varianza de los estimadores.
2. **Eliminación de variables:** Suprimir la variable $X_i$ que causa el conflicto (riesgo de error de especificación).
3. **Combinación de datos:** Mezclar series de tiempo con datos de corte transversal.

### 2.2. Transformación por Primeras Diferencias

Se utiliza para hacer las variables estacionarias, aunque se debe tener **cuidado con incurrir en autocorrelación**.

* **Modelo original:** $Y_t = \beta_1 + \beta_2 X_{2t} + \beta_3 X_{3t} + U_t$
* **Modelo transformado:** $\Delta Y_t = \beta_2 \Delta X_{2t} + \beta_3 \Delta X_{3t} + \Delta U_t$

### 2.3. Transformación mediante Alternativa de Razón (Ratios)

Consiste en dividir el modelo original por una de las variables explicativas (ej. $X_5$) para mitigar la heterocedasticidad y la colinealidad:
$$\frac{Y_t}{X_{5t}} = \beta_1 \left(\frac{1}{X_{5t}}\right) + \beta_2 \frac{X_{2t}}{X_{5t}} + \beta_3 \frac{X_{3t}}{X_{5t}} + \dots + \frac{U_t}{X_{5t}}$$

## 3. Análisis del Ejercicio: Sector Manufacturero (Dougherty)

### Modelo (1): Regresión No Restringida (Log-Log)

$$\widehat{\ln Y} = 2.81 - 0.53 \ln K + 0.91 \ln L + 0.047t$$
$ee = (1.38) \quad (0.31) \quad (0.14) \quad (0.021)$
$R^2 = 0.97 \quad F = 189.8$

**Inconsistencias Detectadas:**

* **Contradicción Teórica:** El coeficiente de $\ln K$ es $-0.53$ (Elasticidad negativa).
* **Insignificancia del Capital:** $$t_{calc} = \frac{-0.53}{0.31} = -1.7129$$
    Como $|-1.7129| < 2.086$ (valor crítico), no se rechaza $H_0: \beta_2 = 0$. El capital no es significativo individualmente.

### Verificación Global

Para las variables explicativas:
$$F = \frac{R^2 / (k-1)}{(1-R^2) / (n-k)} = \frac{0.97 / 3}{(1-0.97) / 20} = 215.55$$
Como $215.5 > 3.10$ (F crítico), el modelo es **muy significativo estadísticamente** a pesar de los fallos individuales.

## 4. Imposición de Restricción Teórica: Rendimientos Constantes

Para corregir el signo del capital, se impone la restricción de **Rendimientos Constantes a Escala (RCE)**: $\beta_2 + \beta_3 = 1$.

### Derivación Matemática del Modelo (2)

1. Partimos de: $\ln Y = \beta_1 + \beta_2 \ln K + \beta_3 \ln L + \beta_4 t + u$
2. Sustituimos $\beta_3 = 1 - \beta_2$:
    $$\ln Y = \beta_1 + \beta_2 \ln K + (1 - \beta_2) \ln L + \beta_4 t + u$$
3. Agrupamos logaritmos:
    $$\ln Y - \ln L = \beta_1 + \beta_2 (\ln K - \ln L) + \beta_4 t + u$$
4. Aplicamos propiedad de ratios:
    $$\ln(Y/L) = \beta_1 + \beta_2 \ln(K/L) + \beta_4 t + u$$

**Resultado del Modelo (2):**
$$\widehat{\ln(Y/L)} = -0.11 + 0.11 \ln(K/L) + 0.006t$$
$R^2 = 0.65 \quad F = 19.5$

* **Logro:** El coeficiente del capital ahora es $+0.11$ (teóricamente consistente).
* **Lógica:** Se redujo la correlación al usar la relación capital-trabajo en lugar de niveles brutos.

## 5. Comparabilidad y Pruebas Estadísticas Finales

### 5.1. ¿Son comparables los $R^2$?

**No**, porque la variable dependiente es distinta ($\ln Y$ vs $\ln(Y/L)$). El $R^2$ mide la proporción de la variación total de la dependiente, y sus variaciones totales ($STC$) difieren.

**Método para hacerlos comparables:**

1. Obtener predicciones de $\ln(Y/L)$.
2. Convertir a predicción de $\ln Y$ mediante la identidad: $\widehat{\ln Y} = \widehat{\ln(Y/L)} + \ln L$.
3. Calcular el coeficiente de correlación al cuadrado ($r^2$) entre el $\ln Y$ real y la nueva predicción.

### 5.2. Prueba F de Restricción (Wald)

Para evaluar si la restricción de rendimientos constantes es válida:
$$F = \frac{(RSS_R - RSS_{UR}) / m}{RSS_{UR} / (n - k)}$$
Donde:

* $RSS_R$: Suma de residuos al cuadrado del Modelo (2) - Restringido.
* $RSS_{UR}$: Suma de residuos al cuadrado del Modelo (1) - No Restringido.
* $m$: Número de restricciones (1).

### 5.3. Interpretación de la Variable de Tendencia ($t$)

* **Función:** Proxy del progreso tecnológico o cambio técnico.
* **Interpretación en modelo semilogarítmico:** El coeficiente de $t$ multiplicado por 100 indica la tasa de crecimiento porcentual anual.
* **En el modelo (1):** $0.047 \rightarrow 4.7\%$ de crecimiento anual debido al progreso técnico.
