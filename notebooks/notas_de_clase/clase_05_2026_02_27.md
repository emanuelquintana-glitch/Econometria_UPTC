## 1. El Modelo de Regresión Lineal Múltiple (MRLM)

El modelo busca explicar el comportamiento de una variable dependiente $Y$ a través de un conjunto de $k$ variables explicativas $X$.

### 1.1 Representación Algebraica

$$Y_i = \beta_1 + \beta_2 X_{2i} + \beta_3 X_{3i} + \dots + \beta_k X_{ki} + u_i$$

Donde:

* $\beta_1$: Intercepto o término constante.
* $\beta_k$: Coeficientes de regresión parcial (miden el efecto de $X_k$ sobre $Y$ manteniendo todo lo demás constante).
* $u_i$: Término de error estocástico.

### 1.2 Enfoque Matricial (Generalización para $n$ observaciones)

Para ser rigurosos, definimos el modelo en términos de matrices de orden $(n \times k)$:

$$\mathbf{y} = \mathbf{X}\boldsymbol{\beta} + \mathbf{u}$$

Donde:

* $\mathbf{y}$ es un vector de $n \times 1$.
* $\mathbf{X}$ es una matriz de $n \times k$ (la primera columna son 1s si hay intercepto).
* $\boldsymbol{\beta}$ es un vector de parámetros de $k \times 1$.
* $\mathbf{u}$ es un vector de perturbaciones de $n \times 1$.

---

## 2. Estimación por Mínimos Cuadrados Ordinarios (MCO)

El objetivo es minimizar la suma de los residuos al cuadrado ($SRC$):
$$Min \sum \hat{u}_i^2 = \mathbf{\hat{u}}'\mathbf{\hat{u}}$$

### 2.1 El Estimador MCO

Mediante la derivación matricial, llegamos a la ecuación normal:
$$\boldsymbol{\hat{\beta}} = (\mathbf{X}'\mathbf{X})^{-1}\mathbf{X}'\mathbf{y}$$

> **Nota Crítica:** Para que $(\mathbf{X}'\mathbf{X})^{-1}$ exista, la matriz $\mathbf{X}$ debe tener rango completo ($k$), lo que implica que no debe existir **multicolinealidad perfecta**.

---

## 3. Pruebas de Significancia e Hipótesis

Una vez obtenidos los estimadores, debemos verificar si son estadísticamente diferentes de cero.

### 3.1 Prueba de Significancia Individual (Prueba $t$)

Se utiliza para evaluar si una variable específica aporta al modelo.

* **H0:** $\beta_j = 0$ (La variable no es significativa)
* **H1:** $\beta_j \neq 0$ (La variable es significativa)

**Estadístico de prueba:**
$$t = \frac{\hat{\beta}_j}{ee(\hat{\beta}_j)}$$

### 3.2 Prueba de Significancia Global (Prueba $F$)

Evalúa si al menos una de las variables explicativas (excluyendo el intercepto) tiene un efecto real sobre $Y$.

* **H0:** $\beta_2 = \beta_3 = \dots = \beta_k = 0$
* **H1:** Al menos un $\beta_j \neq 0$

**Cálculo del estadístico $F$:**
$$F = \frac{R^2 / (k-1)}{(1 - R^2) / (n-k)}$$

Si $F_{calc} > F_{tabla}$, rechazamos la hipótesis nula; el modelo es globalmente significativo.

---

## 4. Medidas de Bondad de Ajuste

* **Coeficiente de Determinación ($R^2$):** Proporción de la variación total de $Y$ explicada por el modelo.
* **$R^2$ Ajustado ($\bar{R}^2$):** Penaliza la inclusión de variables innecesarias que no aportan significativamente al modelo.
    $$\bar{R}^2 = 1 - (1 - R^2) \frac{n-1}{n-k}$$

## 5. Apéndice: Definiciones Técnicas y Glosario

| Término | Definición |
| :--- | :--- |
| **Multicolinealidad** | Situación donde las variables independientes están altamente correlacionadas entre sí, dificultando la distinción de sus efectos individuales. |
| **Homocedasticidad** | Supuesto de que la varianza de los errores es constante para todas las observaciones ($Var(u_i) = \sigma^2$). |
| **Grados de Libertad** | Número de observaciones menos el número de parámetros estimados ($n - k$). |
| **Error Estándar (ee)** | Desviación estándar de la distribución muestral de un estimador. |
| **P-valor (p-value)** | Nivel de significancia más bajo al cual la hipótesis nula puede ser rechazada. Si $p < 0.05$, usualmente se rechaza H0. |
| **Teorema de Gauss-Markov** | Establece que, bajo ciertos supuestos, los estimadores MCO son los Mejores Estimadores Lineales e Insesgados (MELI o BLUE). |

## 7. Derivación Rigurosa del Estimador MCO (Enfoque Matricial)

Para profundizar, debemos entender de dónde sale $\hat{\beta}$. Partimos del vector de residuos $\mathbf{\hat{u}}$:

$$\mathbf{\hat{u}} = \mathbf{y} - \mathbf{X}\hat{\boldsymbol{\beta}}$$

Queremos minimizar la Suma de los Residuos al Cuadrado ($SRC$), que en términos matriciales es el producto escalar del vector de residuos por sí mismo:

$$SRC = \mathbf{\hat{u}}'\mathbf{\hat{u}} = (\mathbf{y} - \mathbf{X}\hat{\boldsymbol{\beta}})'(\mathbf{y} - \mathbf{X}\hat{\boldsymbol{\beta}})$$

Expandiendo la expresión:
$$SRC = \mathbf{y}'\mathbf{y} - 2\hat{\boldsymbol{\beta}}'\mathbf{X}'\mathbf{y} + \hat{\boldsymbol{\beta}}'\mathbf{X}'\mathbf{X}\hat{\boldsymbol{\beta}}$$

### 7.1 Condición de Primer Orden (CPO)

Derivamos respecto a $\hat{\boldsymbol{\beta}}$ e igualamos a cero para encontrar el mínimo:

$$\frac{\partial (SRC)}{\partial \hat{\boldsymbol{\beta}}} = -2\mathbf{X}'\mathbf{y} + 2\mathbf{X}'\mathbf{X}\hat{\boldsymbol{\beta}} = 0$$

Despejando, obtenemos la **Ecuación Normal**:
$$\mathbf{X}'\mathbf{X}\hat{\boldsymbol{\beta}} = \mathbf{X}'\mathbf{y}$$

Y finalmente, aplicando la matriz inversa (asumiendo ausencia de multicolinealidad perfecta):
$$\hat{\boldsymbol{\beta}} = (\mathbf{X}'\mathbf{X})^{-1}\mathbf{X}'\mathbf{y}$$

## 8. Análisis de Varianza (ANOVA) y Descomposición

La variabilidad total de la variable dependiente se puede descomponer de la siguiente manera:

$$STC = SEC + SRC$$

* **STC (Suma Total de Cuadrados):** $\sum (y_i - \bar{y})^2$ - Variación total de Y.
* **SEC (Suma de Cuadrados de la Regresión):** $\sum (\hat{y}_i - \bar{y})^2$ - Variación explicada por el modelo.
* **SRC (Suma de Cuadrados de los Residuos):** $\sum \hat{u}_i^2$ - Variación no explicada (error).

### 8.1 Matriz de Varianza-Covarianza de los Estimadores

La precisión de nuestros $\hat{\beta}$ se mide con esta matriz:

$$Var(\hat{\boldsymbol{\beta}}) = \sigma^2 (\mathbf{X}'\mathbf{X})^{-1}$$

Donde $\sigma^2$ es la varianza del error, estimada como:
$$\hat{\sigma}^2 = \frac{\mathbf{\hat{u}}'\mathbf{\hat{u}}}{n - k}$$

---

## 9. Pruebas de Hipótesis Especiales

### 9.1 Prueba de Restricciones Lineales (Prueba de Wald)

A veces queremos probar si dos coeficientes son iguales (ej: $H_0: \beta_2 = \beta_3$). Para esto usamos el estadístico de Wald o una prueba F restringida:

$$F = \frac{(SRC_R - SRC_{NR}) / m}{SRC_{NR} / (n - k)}$$

Donde:

* $SRC_R$: Suma de residuos del modelo restringido.
* $SRC_{NR}$: Suma de residuos del modelo original (no restringido).
* $m$: Número de restricciones.

## 10. Apéndice Extendido de Definiciones Técnicas

| Concepto | Profundización Matemática / Teórica |
| :--- | :--- |
| **Matriz Identidad ($I$)** | Matriz cuadrada con 1s en la diagonal principal; esencial para definir la homocedasticidad matricialmente: $E[\mathbf{uu}'] = \sigma^2 \mathbf{I}$. |
| **Insesgadez** | Propiedad donde $E[\hat{\beta}] = \beta$. Significa que en promedio, el estimador le atina al valor real del parámetro. |
| **Consistencia** | Propiedad asintótica: a medida que el tamaño de la muestra ($n$) tiende a infinito, el estimador colapsa en el valor real del parámetro. |
| **Matriz de Proyección ($P$)** | Definida como $P = X(X'X)^{-1}X'$. Proyecta el vector $y$ sobre el espacio de las $X$ para obtener $\hat{y}$. |
| **Variable Omisa** | Sesgo que ocurre cuando se excluye una variable relevante que está correlacionada con las incluidas, violando el supuesto de exogeneidad. |
| **Normalidad de Errores** | Supuesto $u \sim N(0, \sigma^2)$ necesario para que las pruebas $t$ y $F$ sean válidas en muestras pequeñas. |

## 11. Resumen de Supuestos de Gauss-Markov (Forma Matricial)

Para que el estimador sea **BLUE (Best Linear Unbiased Estimator)**:

1. **Linealidad:** El modelo es lineal en los parámetros.
2. **Exogeneidad:** $E[\mathbf{u} \vert \mathbf{X}] = 0$ (Los errores no están correlacionados con las X).
3. **No Multicolinealidad Perfecta:** $rank(\mathbf{X}) = k$.
4. **Homocedasticidad y No Autocorrelación:** $Var(\mathbf{u} \vert \mathbf{X}) = \sigma^2 \mathbf{I}$.
