# Econometria_UPTC

Este repositorio es una colección de proyectos, códigos y recursos enfocados en la aplicación práctica de métodos econométricos. Su objetivo es servir como portafolio personal de estudios y herramienta de referencia para el análisis estadístico y la modelización económica utilizando datos reales.
